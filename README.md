# DoDweb3

Static demo for a refreshed Disciples of Dirt website.

Open `index.html` directly or publish the repository with GitHub Pages.

Hero image slots:

- `assets/website-banner-desktop.jpg` for wide desktop screens.
- `assets/website-banner-mobile.jpg` for phone/tablet hero crops.
- `assets/website-banner-square.jpg` for the mid-page story collage.
